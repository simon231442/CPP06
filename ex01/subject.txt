Exercise 01:  Serialization




Implement a class Serializer, which will not be initializable by the user in any way, with the following static methods: #que signifie un serialiseur en cpp?
uintptr_t serialize(Data* ptr);
It takes a pointer and converts it to the unsigned integer type uintptr_t. # quel type de pointeur est pris et quel sont les caracteristique d'un uinptr_t ?
Data*  deserialize(uintptr_t  raw);
It takes an unsigned integer parameter and converts it to a pointer to Data. # c'est quoi les caracteristiques du type Data ?
Write a program to test that your class works as expected.
You must create a non-empty (meaning it has data members) Data structure.
   Use serialize() on the address of the Data object and pass its return value to deserialize(). Then, ensure the return value of deserialize() compares equal to the original pointer.
Do not forget to turn in the files of your Data structure.
